# Linux向けインメモリ超高速音声入力ツール

GPU（NVIDIA RTX 3060など）と `faster-whisper` を使用して、Linux (Ubuntu) 上の**あらゆるウィンドウ（ブラウザ、ターミナル、テキストエディタ等）で動作する**音声入力ツールです。

## 主な特長

- **インメモリ音声処理**: 録音した音声データを一時ファイルに一切書き出さず、メモリ上の NumPy 配列として保持し、直接 Whisper モデルへ入力するため極めてクリーンで高速です。
- **GPU (CUDA) による超高速推論**: 日本語の認識精度が最も高い `large-v3` モデルを使用しながらも、GPU アクセラレーションにより約 0.4〜0.6 秒という一瞬で文字起こしを完了します。
- **動的リンクによるディスク空き容量節約**: システム上にすでに存在する Ollama などの CUDA 12 共有ライブラリ（`libcublas.so.12` 等）を自動検出して実行時にリンクするため、大容量の PyTorch/CUDA 関連の pip パッケージを別途インストールする必要がなく、ディスク容量を圧迫しません。
- **クリップボード連携**: 文字起こし結果を自動でクリップボードにコピーした上で、`Ctrl + V` をシミュレートしてカーソル位置に自動入力します。

---

## 動作環境

- **OS**: Linux Ubuntu (X11 デスクトップ環境を推奨)
- **GPU**: NVIDIA GPU (CUDA 12 互換ライブラリ等がインストールされていること)
- **マイク**: デフォルトの録音入力デバイスが設定されていること

---

## 使い方

1. テキストを入力したい場所（ブラウザの検索バー、Discord、ターミナルなど）に**カーソルを合わせます**。
2. キーボードの **`F8`** キーを押します。
   - デスクトップ通知で「🔴 録音中... text」と表示され、マイクからの録音が始まります。
3. マイクに向かって話します。
4. 話し終わったら、もう一度 **`F8`** キーを押します。
   - デスクトップ通知で「⏳ 認識中...」と表示され、一瞬でテキストがカーソル位置に入力されます。
   - 文字起こしされたテキストはクリップボードにコピーされているため、別の場所で `Ctrl + V` で再貼り付けすることも可能です。

---

## インストール方法

リポジトリをクローン後、以下のセットアップスクリプトを走らせるだけで、システムパッケージの確認、Python環境の構築、自動起動（常駐化）の設定、初回のバックグラウンド起動までがすべて自動で行われます。

```bash
# セットアップスクリプトの実行
bash install.sh
```

### 自動起動（常駐化）の仕組み

スクリプトを実行すると、`~/.config/autostart/` に自動起動用の `.desktop` ファイルが生成されて配置されます。これにより、次回以降はPCを起動（ログイン）するだけで、自動的にバックグラウンドでツールが待機状態になります。
